## 軟體簡介

**BioXAK（Biology X Analysis Kit）** 是一款免費、開放原始碼的 Windows 桌面生物資訊分析軟體，將多種生物資訊分析流程整合於單一平台，專為無程式設計背景的濕實驗室研究人員設計。

無需安裝任何程式環境，下載解壓縮後即可直接執行，並以 MIT 授權條款免費提供於 https://github.com/bowyung/BioXAK/releases/latest 。

---

## 功能模組

| 模組 | 說明 |
|------|------|
| 🧬 **TCGA 癌症分析** | 使用 DESeq2 標準化數據進行差異表現、存活分析與多組比較，自動選擇適當統計方法 |
| ✂️ **CRISPR sgRNA 設計** | 自動生成 sgRNA 候選序列，並提供脫靶評分與 GC 含量分析 |
| 💧 **流式細胞術分析** | 支援 FCS 檔案匯入、雙指數縮放、補償矩陣與互動式閘門設定 |
| 🔬 **引子設計** | 引子對設計，含 Tm 值計算、GC 含量及二級結構檢查 |
| 🔎 **限制酶分析** | 整合 REBASE 資料庫的限制酶切位點映射 |
| 🧮 **分子生物計算機** | DNA/RNA/蛋白質工具，包含分子量計算、密碼子分析與序列轉換 |

## 下載與安裝

1. 前往 [**Releases 頁面**](https://github.com/bowyung/BioXAK/releases/latest)
2. 下載 `BioXAK-v1.0.zip`
3. 解壓縮至任意資料夾
4. 雙擊 `BioXAK.exe` 即可啟動

> **系統需求：** Windows 10 / 11（64 位元），無需額外安裝任何環境。
